# 🧩 Steganography

## 📌 Overview
Steganography is the practice of hiding secret data inside files such as images, audio, or text.

---

## 🎯 Objective
- Understand how hidden data is embedded
- Learn common techniques
- Explore detection methods

---

## 🧠 What is Steganography?

Unlike encryption, steganography hides the **existence of data**, not just its content.

Example:
- Image looks normal
- But contains hidden message

---

## ⚠️ Use Cases
- Secret communication
- Data exfiltration
- Malware hiding payloads

---

## 🔬 Real-World Relevance
Attackers use steganography to hide:
- malware
- command & control data

---

## 🧾 Key Takeaways
- Hard to detect if done properly
- Often combined with encryption
