# 🛠️ Steganography Tools

## 🔍 Steghide

```bash
steghide extract -sf image.jpg
```

---

## 🔍 zsteg

```bash
zsteg image.png
```

---

## 🔍 binwalk

```bash
binwalk image.png
```

---

## 🔍 ExifTool

```bash
exiftool image.jpg
```

---

## 🧠 Key Takeaways
- Tools help detect hidden data
- Multiple tools may be needed
