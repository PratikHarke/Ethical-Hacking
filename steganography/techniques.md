# 🧠 Steganography Techniques

## 🖼️ 1. LSB (Least Significant Bit)
- Modifies smallest bits in image pixels
- Changes are not visible to human eye

---

## 📄 2. Text Steganography
- Hiding data in spaces, formatting

---

## 🔊 3. Audio Steganography
- Embeds data in sound waves

---

## 🧠 4. Image Metadata
- Hiding data in EXIF metadata

---

## 🧾 Key Takeaways
- LSB is most common technique
- Metadata is often overlooked
