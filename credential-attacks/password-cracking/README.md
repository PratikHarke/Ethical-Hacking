# 🔐 Password Cracking

## 📌 Overview
Password cracking is the process of recovering passwords from stored or transmitted data using various techniques.

---

## 🎯 Objective
- Understand password cracking methods
- Learn how attackers break passwords
- Explore tools used in real-world scenarios

---

## 🧠 Why It Matters
Weak passwords can lead to:
- Unauthorized access
- Data breaches
- System compromise

---

## 🔬 Real-World Relevance
Password attacks are one of the most common attack vectors used by attackers.

---

## 🧾 Key Takeaways
- Strong passwords are critical
- Hashing alone is not enough without salting
