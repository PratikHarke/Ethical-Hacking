# 🧠 Password Cracking Techniques

## 🔓 1. Brute Force Attack
- Tries all possible combinations
- Very slow but guaranteed success

---

## 📖 2. Dictionary Attack
- Uses a predefined wordlist
- Faster than brute force

---

## 🌈 3. Rainbow Table Attack
- Uses precomputed hash tables
- Faster lookup for known hashes

---

## 🧪 4. Hybrid Attack
- Combines dictionary + brute force
- Example:
```
password123
admin@2024
```

---

## 🔐 5. Credential Stuffing
- Uses leaked credentials on other sites

---

## ⚠️ Weak Password Examples
- 123456
- password
- admin

---

## 🧠 Key Takeaways
- Password complexity matters
- Attack speed depends on method used
