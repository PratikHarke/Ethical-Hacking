# 📡 DNS Lookup and Vulnerability Analysis

## 📌 Overview
DNS (Domain Name System) translates domain names into IP addresses. Attackers use DNS information for reconnaissance and identifying potential vulnerabilities.

---

## 🎯 Objective
- Understand DNS functionality
- Perform DNS lookup and enumeration
- Identify potential vulnerabilities

---

## 🧠 What is DNS?

DNS resolves domain names into IP addresses:

Example:
```
google.com → 142.250.x.x
```

---

## 🔬 Why Attackers Use DNS
- Discover subdomains
- Identify servers
- Gather infrastructure details

---

## ⚠️ Impact
- Exposure of sensitive infrastructure
- Increased attack surface
- Target identification

---

## 🧾 Key Takeaways
- DNS is critical for reconnaissance
- Misconfigurations can expose systems
