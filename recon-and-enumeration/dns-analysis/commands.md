# 🛠️ DNS Commands

## 🔍 nslookup

```bash
nslookup example.com
```

👉 Retrieves IP address of domain

---

## 🔍 dig

```bash
dig example.com
```

👉 Provides detailed DNS records

---

## 🔍 whois

```bash
whois example.com
```

👉 Retrieves domain registration details

---

## 🔍 host

```bash
host example.com
```

👉 Simple DNS lookup tool

---

## 🔍 Subdomain Enumeration (Example)

```bash
theHarvester -d example.com -b google
```

👉 Finds subdomains and emails

---

## 🧠 Key Takeaways
- These tools help gather target information
- Used in reconnaissance phase
